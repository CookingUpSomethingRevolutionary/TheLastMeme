# The Last Meme
## Description 
After seeing a YouTube video of one of my favorite YouTubers doing this, I decided to try it for myself! I wanted to theme it to something that was funny but also resembled a real Star Wars intro. That's where The Last Meme comes in! I love dystopian and sci-fi books and memes, so I combined my interests to make this project. It took longer than it should have and a couple of CSS tutorials, but I did it! Finally!
